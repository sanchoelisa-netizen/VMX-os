# Templates

Modelos reutilizáveis. Nunca crie um documento do zero se houver um template aqui.

- `diagnostico-geral.md`, `diagnostico-empresarial.md`, `diagnostico-comercial.md`, `matriz-diagnostico.md` — diagnóstico
- `plano-acao.md` — plano de ação
- `proposta-comercial.md`, `proposta-consultoria.md` — propostas
- `checklist-contrato.md`, `checklist-implantacao.md` — checklists operacionais
- `email-boas-vindas.md` — comunicação com cliente
- `landing-page.md` — estrutura obrigatória de landing page
- `checklist-qualidade-entrega.md` — checklist de qualidade de qualquer entrega
