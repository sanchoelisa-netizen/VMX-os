# Playbook - Onboarding de Cliente

## Objetivo
Padronizar o passo a passo para iniciar novos clientes e novos projetos.

## Processo
1. Receber contrato assinado
2. Criar projeto
3. Agendar kickoff
4. Configurar CRM
5. Coletar acessos
6. Definir responsáveis

## Checklist
- Contrato
- Kickoff
- Acessos
- CRM
- Responsáveis definidos
